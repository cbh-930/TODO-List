# TODO List 前端应用

基于 React + TypeScript + Vite 构建的现代化前端应用。

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 注意事项

确保后端服务在 `http://localhost:8080` 运行，否则前端无法正常获取数据。
